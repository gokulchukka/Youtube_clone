# 🎬 YouTube Clone

A modern YouTube Clone built using **React.js** that allows users to browse videos, search content, and watch videos with a clean responsive UI.

---

## 🚀 Features

- 🔍 Search YouTube videos
- 📺 Watch videos with embedded player
- 🏠 Home page with trending videos
- 📱 Responsive design
- ⚡ Fast React-based UI
- 🎨 Modern styling

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- CSS
- YouTube Data API
- Axios
